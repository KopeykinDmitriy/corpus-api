**API для сбора данных**

## Описание

REST API для сбора данных из различных источников с возможностью экспорта в формате JSONL.

## Возможности

- **Сбор данных из внешнего API** - получение данных из JSON API с преобразованием в структурированный формат
- **Сбор данных с веб-страниц** - извлечение текстового контента с сайтов с поддержкой обхода внутренних ссылок
- **Обработка файлов** - поддержка форматов: TXT, JSON, JSONL, CSV и других текстовых файлов
- **Экспорт данных** - автоматическое сохранение результатов в JSONL файл для скачивания

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/KopeykinDmitriy/corpus-api.git
cd project
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```
